Foundations, recapped.

The system is built on Fraunces + Inter + JetBrains Mono, a cream-and-ink foundation, magenta as the single brand colour, and five marketing accents confined to the landing's service grid. Notched corners appear only on hero composition, status hero, code chips, marketing blocks, dark quotes, and the gradient closing band.
