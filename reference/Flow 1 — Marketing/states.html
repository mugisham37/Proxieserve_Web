The landing page is server-rendered, so its hero text is always present. Loading, empty, error, offline, and edge-case states sit underneath — here's each one in isolation.

Implementation. A single @media (prefers-reduced-motion: reduce) block in tokens.css overrides all transitions to 0.01ms. No bespoke motion in JS — we let CSS rules win.
