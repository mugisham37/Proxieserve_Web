FLOW 10 / CROSS-CUTTING SYSTEMS · 7 SPECS

The connective tissue.

The systems that aren't a screen but touch every screen: installability, consent, internationalization, theming, print, transparency, and first-run guidance. Each is shown as a working artifact plus its rule.

01 / PWA — installable app

Add to home screen

Offline-first

Install ProxiServe

Add to your home screen for faster access and updates even offline.

A real app, no store. Web App Manifest + service worker make ProxiServe installable on Android (and iOS via Add to Home Screen). The prompt appears after the user has had one successful interaction — never on first load. Caches the shell + last-seen data for offline use.
02 / Cookie consent

Consent, not a wall

Privacy

Cookies, briefly: essential ones keep you signed in and remember your tracking code. Optional analytics help us improve. Your call.

Banner, never a blocker. Doesn't obscure content; doesn't pre-tick optional categories. "Customise" opens per-category toggles (Essential / Analytics — Marketing is absent because we don't do ad-tracking). Choice persisted and revocable from the footer. Compliant with Rwanda's data-protection law + GDPR for diaspora users.

Application codes, phone numbers, and amounts stay LTR (they're not language). Today's languages (EN/RW/FR) are LTR; the system is ready for RTL markets without a rebuild.
04 / Dark mode — token-driven

One system, two themes

Theming
Light · public + client default
Hand it over.
Dark · agent + admin default
Hand it over.
Swap surfaces, hold the brand. Dark mode is a data-theme="dark" token swap — surface + ink invert, but the magenta brand and the status colours stay constant for instant recognition. Public/client zones default light; agent/admin default dark (set in F9). Per-user override always wins; respects prefers-color-scheme on first visit.
